PRODUCTS

音楽サークル用Webサイト

 GitHub

自身が所属している音楽サークル用に作成したサイト。
サークルの代表が所有しているスタジオの予約管理、イベントの開催などをWebにて取りまとめられるようにするため作成。
フロントエンドはVueJsを用いたSPA、バックエンドはRailsをAPIサーバとして使用し、Dockerコンテナ上で起動。
ユーザ（サークルメンバ）からフィードバックをもらいながらUI/UXの改善、機能の追加・修正等を適宜実施。

使用技術

HTML / SCSS / VueJs / Ruby on Rails / Docker / AWS(EC2, ALB, Route53)

機能一覧

◆ 認証機能

新規ユーザ登録 / ログイン / ログアウト / パスワード変更 / パスワード再設定

  • パスワードはbcryptにて暗号化し保存
  • Vuexにて認証用トークンを保存し、ログイン状態を維持
  • パスワード失念時は登録メールアドレスに再設定用のトークン付URLを送信し、再設定可

    • ◆ スタジオ予約機能

    サークル代表が所持しているスタジオ利用時に予約できる機能

    スタジオ予約、予約キャンセル / スタジオ予約一覧表示 / スタジオ予約確認（管理者用）

  • スタジオ予約が行われた際、LINEのMessaging APIを利用しグループラインへ通知
  • 予約の重複が起こらないようバックエンド側でチェック処理を実装（他日時の整合性等のチェックも実装）

    • ◆ 宿泊機能

    サークル代表の空き部屋（2部屋）へ宿泊時に予約できる機能

    宿泊予約、予約キャンセル / 宿泊予約一覧表示 / 宿泊予約確認（管理者用）

  • 予約する部屋を選択して予約が可能
  • 予約する部屋を選択しない場合、空いている部屋をシステムで判断し予約

    • ◆ イベント機能

    セッション、飲み会等のイベントを開催する機能

    イベント登録、編集、削除 / イベント一覧表示、詳細表示 / イベントへの参加、参加キャンセル / 参加イベント一覧表示 / イベントへのコメント新規投稿、編集、削除

  • イベントが新規登録された際、LINEのMessaging APIを利用しグループラインへ通知
  • セッションイベントの場合、曲の追加や各パートへのエントリーが可能

    • ◆ お知らせ機能

    機能の追加、サークル運用にあたってのお知らせ等を通知するための機能

    お知らせ登録、編集、削除 / お知らせ一覧表示、詳細表示

  • お知らせが新規登録された際、LINEのMessaging APIを利用しグループラインへ通知
